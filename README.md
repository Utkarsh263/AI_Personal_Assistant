# 🤖 AI Personal Assistant

This is a simple AI-based web application that allows users to ask questions and summarize emails using the Groq API.

---

## 🚀 Features

- Ask questions using AI  
- Summarize emails into short bullet points  
- Simple and clean user interface  
- Fast responses using Groq  

---

## 🛠️ Technologies Used

- Python (Flask)  
- HTML, CSS, JavaScript  
- Groq API  
- python-dotenv  

---

## ⚙️ How to Run

1. Install required libraries:
```bash
pip install flask groq python-dotenv
```

2. Create a `.env` file and add your API key:
```env
GROQ_API_KEY=your_api_key_here
```

3. Run the application:
```bash
python app.py
```

4. Open in browser:
```
http://127.0.0.1:5000/
```

---

## 📌 Description

The application has two main functionalities:

- **Ask Anything:** Users can enter any query and get AI-generated responses.  
- **Email Summarizer:** Users can paste email content and receive a concise summary.  

---

## 📄 Conclusion

This project demonstrates the integration of AI APIs with a web application using Flask, enabling real-time interaction and text processing.
